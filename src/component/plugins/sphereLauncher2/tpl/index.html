Скомпилировать Лаунчер

В этом разделе вы можете скомпилировать Sphere Launcher.

Перейти

Создать конфигурацию launcher.json

Файл необходим для того чтоб Sphere Launcher знал с какого источника загружать файлы игры.

Этот файл необходимо будет загрузить на Ваш FTP.

Перейти
Вопрос / Ответ

Скомпилируйте себе лаунчер, запустите его и чтоб открыть страницу создания архивов игры, нажмите в лаунчере F1.

Да, на странице создания архивов есть возможность подключиться к FTP указать директорию сохранения.
Архивы будут загружены на FTP после создания их.
Если вы обновляете уже папку с архивом, тогда будут загружаться только новые (измененные) архивы на FTP.

Файл archive.csv создается после создания архивов, в нем содержится реестр файлов, их размер, хэш.
Сфера Лаунчер использует для проверки актуальности файлов у пользователя именно этот файл.
Ссылка на него указывается в launcher.json.

Лаунчер входит в стандартную лицензию SphereWeb и будет активирован до даты окончания лицензии.

Если у вас нет платной подписки на SphereWeb, тогда лаунчер будет активирован на несколько дней.

Проще говоря: если лицензия Сферы заканчивается 17.08.2025, тогда лаунчер будет активирован до 17.08.2025. Дата окончания лаунчера будет зашита в само приложение.

  • Даже если вы пополните баланс Сферы, это не повлияет на дату окончания лаунчера у конечных пользователей.
  • В таких случаях необходимо будет перекомпилировать лаунчер и заменить его на новый.
  • Через автообновление лаунчер обновится у пользователей автоматически.

Двумя путями:
  • Купить лицензию в Сфере (12$ / 20$), автоматически вы сможете использовать лаунчер.
  • Купить отдельно лаунчер на неограниченный срок (100 USDT).

Для покупки свяжитесь с разработчиком SphereWeb.

После оплаты лицензии, вы получите ссылку на скачивание лаунчера и инструкцию по активации.
